# Sales & Profit Analysis using Excel

 ## Project Overview
This is a beginner-level data analysis project using Microsoft Excel. 
I analyzed regional sales and profit data of a store to generate business insights. 
Pivot Tables and Pivot Charts were used to summarize and visualize the data.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts

## Key Insights
- West region generated the highest sales and profit.
- Central region had comparatively lower profit.
- Simple visualizations were created to quickly understand business performance.

## Skills Demonstrated
- Data cleaning and summarization
- Business insight generation
- Data visualization
